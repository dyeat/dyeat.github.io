```shell
-mtime 搜尋檔案的修改時間(天)
-mmin 搜尋檔案的修改時間(分鐘)
-ctime 搜尋檔案的建立時間(天)
-cmin 搜尋檔案的建立時間(分鐘)
-atime 搜尋檔案的最後開啟時間(天)
-amin 搜尋檔案的最後開啟時間(分鐘)
```

### 範例

```shell 
find [路徑] [選項] [參數]
find ./ -mtime 3 # 在當前目錄下搜尋3天時修改的檔案
find ./ -mtime +3 # 在當前目錄下搜尋3天前修改的檔案
find ./ -mtime -3 # 在當前目錄下搜尋3天內修改的檔案
```
